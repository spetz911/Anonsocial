Кроме того, что обладают всеми правами \textbf{Зарегистрированных журналистов}.		\begin{itemize}		    \item Создавать аккаунты журналистов.		    \item Создавать аккаунты VIP-гостей.		    \item Редактировать статьи и комментарии к ним.		  %  \item Создавать, редактировать, удалять новости.		\end{itemize}    \item Админа.		\begin{itemize}		    \item Создаёт аккаунты модераторов.		\end{itemize}\end{itemize}\pagebreak;\section{UseCase-диаграмма}(Диаграмма создана в системе Enterprise Architect)\begin{center}\includegraphics[width=0.8\textwidth]{pdf/soc_net.pdf} \end{center}\pagebreak;\section{Интерфейс}Мы не будем здесь приводить требования к интерфейсу системы, т.к.\begin{itemize}    \item если не обращать внимания на художественные изыски, то все социальные сети выглядят одинаково    \item понятие об интерфейсе следует из описания выше    \item это выходит за рамки данных лабораторных работ 			и относится скорее к проектированию клиента, чем всей системы.\end{itemize}
